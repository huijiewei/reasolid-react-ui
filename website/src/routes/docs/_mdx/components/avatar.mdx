---
title: 头像
description: 头像组件用于表示用户，显示个人资料图片、首字母或备用内容。
---

## 导入

```js
import {
  Avatar,
  AvatarImage,
  AvatarFallback,
  AvatarIndicator,
  AvatarGroup,
} from "@resolid/react-ui";
```

- `Avatar`: 头像组件的根容器。
- `AvatarImage`: 代表用户的图片。
- `AvatarFallback`: 一个包装器，其内容在图像未能加载时显示。
- `AvatarIndicator`: 一个包装器，其内容显示为头像的装饰。
- `AvatarGroup`: 一个用于将多个头像堆叠在一起的包装器。

## 举例

### 演示

```jsx demo
import { Avatar, AvatarImage, AvatarFallback } from "@resolid/react-ui";

export default function App() {
  return (

        <AvatarFallback />
      </Avatar>
    </div>
  );
}
```
